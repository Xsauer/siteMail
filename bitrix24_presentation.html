The Pipeline
Воронка

5 stages. Zero friction.

5 стадий. Ноль лишних действий.

Every transition is triggered automatically. The manager only needs to create the deal and add products.

Каждый переход запускается автоматически. Менеджеру достаточно создать сделку и добавить товары.

📥
New Deal
Новая сделка
Manager creates a deal, or one is created automatically from connected OLs, and adds products and prices. Without leads, it goes straight to the funnel.
Менеджер создаёт сделку, либо она создается автоматически с подключенных ОЛ и добавляет товары и цены. Без лидов — сразу в воронку.
📧
Quote Sent
КП отправлено
CRM robot automatically sends a branded email from the manager's address with a unique link to the quote page.
Робот CRM автоматически отправляет фирменное письмо с адреса менеджера с уникальной ссылкой на страницу КП.
⚡ Automatic ⚡ Автоматически
✅
Quote Approved
КП принято
Client opens the page, reviews the product list, downloads PDF if needed, and clicks "Approve". One webhook call — the deal moves itself to the next stage, a confirmation email is logged, and a task is created for the team.
Клиент открывает страницу, просматривает список товаров, скачивает PDF при необходимости и нажимает «Подтверждаю» (Akceptuję). Один webhook — сделка сама переходит на следующую стадию, письмо фиксируется в таймлайне, задача создаётся автоматически.
⚡ Automatic ⚡ Автоматически
💳
Deposit via Stripe
Депозит через Stripe
Immediately after approval, the system generates a Stripe payment link and sends it to the client. No manual invoicing.
Сразу после акцепта система генерирует ссылку на оплату в Stripe и отправляет клиенту. Никаких ручных счетов.
⚡ Automatic ⚡ Автоматически
🏁
In Progress → Closed
В работе → Закрыто
After deposit is received, the deal moves to execution. The team gets a task with all details. Manager tracks progress in Kanban.
После оплаты депозита сделка переходит в работу. Команда получает задачу с деталями. Менеджер отслеживает прогресс в Kanban.
Quote Page
Страница КП

What the client sees

Что видит клиент

A clean, mobile-friendly page — no login, no forms, no confusion. The data is pulled live from Bitrix24.

Чистая мобильная страница — без логина, форм и лишнего. Данные подгружаются из Bitrix24 в реальном времени.

📋
Product table
Таблица товаров
Name, quantity, unit price and total — pulled directly from the deal's product rows.
Наименование, количество, цена и сумма — берётся напрямую из позиций сделки.
⬇️
PDF download
Скачать PDF
Client can download a branded PDF quote — generated in the browser, no server needed.
Клиент может скачать фирменный PDF — генерируется прямо в браузере, без сервера.
✅
One-click approval
Акцепт одним кликом
No SMS codes, no e-signature flows. One button — approval is logged instantly in CRM.
Никаких SMS-кодов и подписей. Одна кнопка — акцепт мгновенно фиксируется в CRM.
👁️
Open tracking
Трекинг открытия
When the client opens the page, a webhook fires and logs "Quote viewed" in the deal timeline.
Когда клиент открывает страницу — вебхук фиксирует «КП просмотрено» в таймлайне сделки.
